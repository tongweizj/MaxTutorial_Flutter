# 创建一个 Dart 项目

## SETP1  Get more command-line developer tools

```bash
pub global activate stagehand
```

## 4. Create a small app

```bash
dart create -t console-full cli
```

## 5. Get the app’s dependencies

```dart
 pub get
```

## Run the app

```bash
dart bin/cli.dart

```
